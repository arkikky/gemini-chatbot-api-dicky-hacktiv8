# Chatbot AI Gemini

## Deskripsi

Aplikasi chatbot berbasis web sederhana yang menggunakan Google Gemini API untuk menghasilkan respons. Aplikasi ini memiliki antarmuka pengguna yang simpel dan bersih.

## Fitur

- Obrolan real-time dengan bot.
- Antarmuka yang ramah pengguna.
- Mudah untuk di-setup dan dijalankan.

## Instalasi

1. Clone repositori:
   ```bash
   git clone <url-repositori>
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd gemini-chatbot-api-dicky-hacktiv8
   ```
3. Instal dependensi:
   ```bash
   npm install
   ```
4. Buat file `.env` dan tambahkan kunci API Gemini Anda:
   ```
   GEMINI_API_KEY=kunci_api_anda
   ```

## Cara Menjalankan

1. Jalankan server:
   ```bash
   node index.js
   ```
2. Buka browser Anda dan kunjungi `http://localhost:3000`
